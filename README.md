# Cisco Introduction to Cybersecurity

This repository contains my notes and certificate from Cisco's Introduction to Cybersecurity course.

## Course Overview

This course introduced the fundamental concepts of cybersecurity, including:

- Cybersecurity principles
- Common cyber threats
- Types of malware
- Network security basics
- CIA Triad
- Cybersecurity careers
- Risk management
- Data protection

## Skills Learned

- Understanding cyber threats and attack vectors
- Basic security concepts
- Confidentiality, Integrity and Availability (CIA Triad)
- Password security and authentication
- Social engineering awareness
- Introduction to network security
